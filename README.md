# Poke

This App is Pokemon Picture Book using [create-next-app with electron-typescript](https://github.com/vercel/next.js/tree/canary/examples/with-electron-typescript) + [GraphQL Pok√©mon](https://github.com/lucasbento/graphql-pokemon)

## How to use

```bash
$ npm run dev
```

Available commands:

```bash
"build-renderer": build and transpile Next.js layer
"build-electron": transpile electron layer
"build": build both layers
"dev": start dev version
"dist": create production electron build
"type-check": check TypeScript in project
```

## Notes

You can create the production app using `npm run dist`.
